# 普通流

上到下，左到右，行内，块级

# 浮动

按照普通流出现，然后尽可能向左向右偏移

# 绝对定位

脱离普通流，不会对兄弟节点造成影响

# BFC

属于普通流，一个独立的渲染区域，隔绝外界，BFC之间互不影响。

# BFC形成

根元素，浮动元素，绝对定位元素，overflow不可见元素

# BFC作用：

  1. 避免外边距重叠
  2. 清除浮动，子元素浮动后父元素会塌陷，父元素使用BFC避免这一点
  3. 阻止元素被浮动元素覆盖

# 居中

# 行内元素：

  1. text-align:center
  2. line-height 和 height

# 块级元素：

  1. margin：0 auto，水平自适应。auto必须在元素拥有足够的空间时才能生效

# 水平垂直居中

  1. 子绝父相，top和left都是calc（50%-height和width）
  2. 定位+transform：同样topleft50%，然后transform: translate(-50%, -50%)
  3. 定位+margin：上下左右都是0，然后margin auto，
  4. flex-box：justify-content和align-items都是center

# 两栏布局

float+margin

float+bfc

flex

# 三栏布局

  1. float+margin
  2. float+bfc
  3. flex
  4. display:table, table-cell
  5. 定位
  6. 圣杯布局：float-left，主元素100%。margin-left：width和100%；position和left；

# 1、盒模型

dom元素采用的布局模型，分为content-box和boreder-box

宽高的计算方式：content-box的width只包含content，border-box的width包含content，padding、border

# 2、BFC

块级格式化上下文，独立的渲染区域，隔离BFC和外界元素

触发：根元素、position：绝对\固定、float、overflow不可见

规则：

BFC内两个Box垂直排列，margin重叠；

BFC区域不会和float元素区域重叠

BFC计算高度时，会计算float元素

文字不会被float覆盖，而是环绕在周围

应用：

阻止margin重叠

清除内部浮动：原理就是让两个div在同一个BFC里

自适应两栏布局

阻止元素被浮动元素覆盖

# 3、层叠上下文

bg < z-index负数 < 块级 < 浮动 < 行内 < z-index正数

# 4、居中布局

水平居中：块级margin: 0 auto； 普通flex + justify-content: center；

垂直居中：line-height: height；flex + align-items: center；

# 5、选择器优先级

important > 行内 > id > class > tag > * > 继承

# 6、去除浮动

增加尾元素、父级BFC、父级设置高度

# 7、link和@import

link还能定义RSS、Rel、@import只能加载css

link会同步加载、@import等页面加载完再加载

嵌套、变量、循环条件语句等
