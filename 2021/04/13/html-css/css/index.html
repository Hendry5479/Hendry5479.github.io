Publish Date:   2021-04-13

css3

  • css3增加的大量样式、动画、3D特效和移动端特性
  • css的使用方式
    • 内嵌style
    • 外链link
      • link rel=”stylesheet” href=”…”
    • 导入style @import(url(xxx))
    • 行内式，一边是后台调试使用

选择器

  • 标签选择器：常用于标签初始化
  • id选择器：# 前缀 #id
  • class选择器 . 前缀 .class
  • 原子类：将所有的常用字号、文字颜色、行高、外边距、内边距等都设置为单独的类
  • 复合选择器 li.class
  • 后代选择器：空格表示后代 .box p{}
  • 交集选择器：h3.spec
  • 并集选择器：ul, ol {}
  • 伪类是添加到选择器的描述性词语，指定元素的特殊状态
    • 超级链接拥有4个特殊状态
    • a:link 未访问
    • a:visited 已访问
    • a:hover 悬停
    • a:active 按下按键但是还没有松开按键

权重计算

  • id > class > 标签

CSS3新增选择器

  • .box>p {} 直接后代元素 子选择器
  • img+span {} 相邻兄弟选择器
  • h3~span {} 通用兄弟选择器
  • a~b {} 选择a之后所有同级的b元素
  • 序号选择器
    • :first-child
    • :nth-child(an+b) 匹配属于其父元素的第 N 个子元素，不论元素的类型
      • 2n+1 == odd 2, n == even
    • :nth-of-type(3) 第三个某特定类型的元素
    • :nth-last-of-type(n)
  • 属性选择器
    img[alt=”xxx”]，带有alt属性的img标签

新增伪类

::伪元素

  • 虚拟动态创建的元素
  • ::before，创建一个成为匹配的元素的第一个子元素，必须设置content
  • ::after
  • ::selection 应用于被用户高亮的部分，用鼠标圈中的部分
  • ::first-letter：选中某元素的第一个字母（必须是块级元素）
  • ::first-line：第一行

color

  • 十六进制：黑色是#000，白色是#fff，常见的灰色有#ccc、#333、#2f2f2f等
  • rgb()： color: rgb(255, 0, 0);
  • rgba()：最后一个参数表示透明度，0纯透明，1纯实心

font-size

  • 网页正文字号通常是16px，浏览器最小支持10px
  • font-weight，通常用normal和bold
  • font-style，设置字体倾斜
  • text-decoration，设置文本修饰线外观的（下划线、删除线）
  • text-indent，定义首行文本的缩进量，缩进两个字符写作2em; em表示字符宽度
  • line-height，定义行高，行高 = 字体 + 间隙
    • line-height: 30px;
    • line-height: 1.5; 字号的1.5倍
  • 另外，单行文本垂直居中：text-align: center;

font合写属性

  • font作为style, weight,size, height，family的合写
  • font: italic bold 20px/1.5 Arial, “微软雅黑”;

继承性

  • 文本相关的属性具有继承性，只需要给祖先标签设置，即可在后代标签中生效
  • 通常会设置body标签的字号、颜色、行高等，当做默认样式
  • 在继承的情况下，选择器权重计算失效，而是“就近原则” 谁描述得最靠近内容，听谁的

字体属性

  • 字体可以是列表形式，一般英语字体放到前面，后面的字体是前面字体的“后备”
  @font-face {
    font-family: '字体名称';
    font-display: swap;
    src: url('eot字体文件地址');
    src: url('eot字体文件地址') format('embedded-opentype'),
    url('woff2字体文件地址') format('woff2'),
    url('woff字体文件地址') format('woff'),
    url('ttf字体文件地址') format('truetype'),
    url('svg字体文件地址') format('svg');
    }

盒模型

  • width/height + padding + border
  • width单位是px，移动端涉及百分数、rem等单位
  • 块级元素没有设置width时，会自动撑满，但不代表width会继承
  • height如果不设置，会自动被内容撑开，没有内容，默认是0
  • 小属性层叠大属性
  • margin
    • 盒子之间的距离
    • 开发时需要清除默认margin
    • 盒子的水平居中
      • margin设置为auto将水平居中
      • 文本居中text-align和盒子居中是两个概念
      • 盒子的垂直居中需要使用绝对定位
  • 块级元素不能并排显示，可设置宽高，width自动撑满，如div，section，header，li
  • 行内元素可以并排，不可设置宽高

行内块

  • img和表单元素
    即可设置宽高，也可并排显示

行内和块级的转换

  • display:block, inline(少见), inline-block
  • display:none，将元素隐藏，放弃位置
  • visibility：hidden是隐藏，但仍有位置
  • box-sizing： border-box;
    • padding和boder将占据宽高，将用于移动网页制作

浮动

  • float:left;
  • 浮动使用要点：要浮动，并排的盒子都要设置浮动。只有并排元素需要设置浮动，垂直的不需要
  • 父盒子要有足够的宽度，否则子盒子会掉下去
    +
  • 子盒子会按顺序进行贴靠，如果没有足够空间，则会寻找再前一个兄弟元素
  • 浮动的元素不再区分块级、行内元素，已经脱离了标准文档流，一律能够设置宽度和高度

BFC规范，Box Formatting Context

  • 块级格式化上下文，独立容器
  • 盒子不设height，子元素浮动脱离，无法撑起自身，没有形成BFC
  • BFC取消margin竖直塌陷
    • margin塌陷：在标准文档流中，竖直方向的margin会出现叠加现象，即大margin会覆盖掉小margin，竖直方向的两个盒子中间只有一个较大的margin
    • margin的塌陷现象分两种情况：
      1，兄弟关系的盒子margin塌陷
      给上面的盒子加个margin-bottom（这个距离等于两个盒子之间的距离）
      或者给下面的盒子加个margin-top（这个距离等于两个盒子之间的距离）
      2 ，父子关系的盒子margin塌陷
      给父盒子添加上下边框或者上下padding
  • BFC可以阻止元素被浮动元素覆盖

清除浮动

  • 浮动一定要封闭到盒子中，否则会对页面后续元素产生影响

1.让浮动的父盒子成为BFC，最好的方法是overflow:hidden
2.给后面的父盒子设置clear:both，clear浮动影响，both表示两边
3.::after给盒子添加最后一个子元素，并设置clear:both

定位

相对定位：相对自己原来的位置

position:relative

  • 相对定位会在老家留坑，本质是原来的位置，只是渲染在新地方，不会对其他元素产生影响
  • 子绝父相：用于微调位置，当作绝对定位的参考盒子

绝对定位

  • 以坐标进行位置精准描述，拥有绝对位置
  • 绝对定位脱离标准文档流，将释放自己的位置，对其他元素不会产生干扰
  • 绝对定位的盒子并不是永远以浏览器作为基准点，而是以自己祖先元素中，离自己最近的拥有定位属性的盒子，当做基准点
    • 这个盒子通常是相对定位的，这个性质也叫作子绝父相
  • 绝对定位的盒子垂直居中
    position: absolute;
    top: 50%;
    margin-top: -height/2;

堆叠顺序z-index属性

  • 数值大的压住数值小的
  • 用途：压盖遮罩，结合精灵图，结合JS实现动画

固定定位

  • 固定定位只能以页面为参考点
  • 脱离标准文档流
  • 固定定位的用途：“返回顶部”、“楼层导航”

总结

  • 脱离标准文档流的方法：浮动、绝对定位、固定定位

边框的三要素

  • border： 1px solid red;
  • 小属性：
    • border-width, border-color,
    • border-style: solid, dashed, dotted
  • 去掉边框：border-left: none;
  • 利用边框制作三角形

border-radius

  • px单位，表示圆角的半径，百分数作单位，50%是圆角

阴影

  • box-shadow: 10px 20px 30px rgba(0,0,0,.4);
  • x偏移，y偏移，模糊量，颜色
  • box-shadow: 10px 20px 30px 40px rgba(0,0,0,.4);
  • 阴影延展
  • box-shadow: inset 10px 20px 30px 40px rgba(0,0,0,.4);
  • 内阴影
  • box-shadow属性值可以用逗号隔开多个，表示携带多个阴影

背景

background-repeat

  • repeat, repeat-x, repeat-y, norepeat
    background-size：width, height; 等比例设置写auto
  • contain和cover
    • contain智能容纳到盒子里，cover将改变尺寸撑满盒子
      background-clip
  • border-box, padding-box, content-box
    background-attachment
  • 决定背景图像的位置是在视口内固定，或者随着包含它的区块滚动
  • fixed 内外滚动条不动
  • local 都动
  • scroll 内不动，外动 默认

background-position

  • CSS精灵可以减少HTTP请求数，加快网页显示速度。缺点是不方便测量、后期改动麻烦

background: white url(images/archer.png) no-repeat center center;

  • 背景位置center center

线性渐变

  • background-image: linear-gradient(to right, blue, red);
    • 渐变方向 开始颜色 结束颜色
    • 渐变方向可以写成度数，45deg
    • 可以有多个颜色值，用百分数定义位置 yellow 20%

      浏览器私有前缀

  • 对试验性质的css属性加标识

径向渐变

  • background-image: radial-gradient(50% 50%, red, blue);
  • 圆心坐标，中心颜色，边缘颜色

旋转变形

  • transform: rotate(45deg); 顺时针方向
  • transform-origin 设置变换原点

    缩放变形

  • transform: scale(3);

    斜切变形

  • transform: skew(10deg, 10deg);

    位移变形

  • transform: translate(100px, 200px);
  • 与相对定位相似，老家留坑，形影分离

    3D旋转

  • transform: rotateX(30deg);

perspective属性

  • 定义透视强度，可以理解为“人眼到舞台的距离”

空间移动

  • transform: rotate(30deg) translateX(30px) translateZ(100px);

transition过渡

  • 作用：添加补间动画

  • transition: width 1s linear 0s;

  • 过渡属性，动画时长，变化速度曲线，延迟时间

  • 所有数值类型的属性都可以参与过渡

  • 背景颜色和文字颜色都可过渡

  • 所有属性参与过渡，写all

  • transition: all 1s linear 0s;

  • 缓动参数

  • 贝塞尔曲线

  • transition: width 1s cubic-bezier(0.1, 0.7, 1.0, 0.1) 0s;

  • https://cubic-bezier.com/

动画

  • 定义动画 @keyframes 关键帧

  • 项目上线前，补上@-webkit-的私有前缀

    @keyframes r {
from {
  transform: rotate(0);
}

to {
  transform: rotate(360deg);
}
}

动画调用

  • 使用animation调用动画
  • animation: r 1s linear 0s 3;
  • 执行次数如果永远执行写infinite
  • 如果让动画的偶数次逆向执行，加上alternate参数
  • 如果让动画停止在结束状态，加上forwards
