HTML

列表：dl > dt > dd

  • 有序列表：ol > li
  • 无序列表：ul > li
  • li不能单独使用, 必须放到ul或者ol中; ul的子标签只能是li
  • li标签是容器，内部可以放任何其他标签

    基本控件

    input

    input type="text" value="xxx" placeholder="xxx" disabled

  • input type=”radio”可以创建单选按钮 (互斥按钮应该设置name为相同值)

  • 单选按钮要有value值，用于向服务器提交

  • 添加checked属性，表示默认被选中

  • label将文字和单选按钮绑定，单击文字等同于点击按钮

    <!-- label不再需要添加for属性 -->
<label>
  <input type="radio">
</label>

  • input type=”checkbox” 复选框

  • input type=”password”

    select 下拉菜单

    <select>
  <option value="a"></option>
  <option value="b"></option>
</select>

    textarea

  • rows cols设置文本框的行数和列数

    表单常见的三种input

表格标签

<table>
  <caption>title</caption>
  <tr>
    <th>row-title</th>
    <td>A</td>
    <td>B</td>
  </tr>
</table>
  • table通常有border属性，以实现边框
  • caption：标题，第一个子元素
  • th是标题小格，代替td作为标题小格
  • colspan和rowspan设置td或者th的列或者行的跨度
  • thead，tbody，tfoot定义表头、核心内容、表尾
