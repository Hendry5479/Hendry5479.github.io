HTML

列表：dl > dt > dd

  • 有序列表：ol > li
  • 无序列表：ul > li
  • li不能单独使用, 必须放到ul或者ol中; ul的子标签只能是li
  • li标签是容器，内部可以放任何其他标签

a标签：

title设置鼠标悬停文本，target=”blank”将在新标签页打开

页面内锚点：给标签设置id，url后接#id将滚动到对应的位置

指向exe, zip, rar等文件格式的链接，将自动成为下载链接

HTML5区块标签

  • span是区块标签，无显示效果
  • figure是独立的引用单元，比如拓展视野的图片等，这部分转移时不影响到主体

    img图片

audio音频

  • controls 显示播放控件
  • autoplay 自动播放
  • loop 循环

video视频

controls

HTML5中新增的表单控件

  • input种类：

  datalist 为输入框提供备选项
    option value=“xxx”

表单创建

  • 表单以form开始
    form action="save.php" method="post"
xxx

    基本控件

    input type="text" value="xxx" placeholder="xxx" disabled
  • input type=”radio” (互斥的单选按钮应该设置name为相同值)
  • 单选按钮要有value值，用于向服务器提交，checked表示默认被选中
  • label将文字和单选按钮绑定，单击文字等同于点击按钮
    • label for=’id’
      • input
  • input type=”checkbox”
  • input type=”password”
  • select 下拉菜单
    • option value
  • textarea
  • rows cols设置文本框的行数和列数

    表格标签

    table
tr
td
  • table通常有border属性
    • caption是标题，第一个子元素
    • th是标题小格，代替td作为标题小格
    • colspan和rowspan设置td或者th的列或者行的跨度
    • thead，tbody，tfoot定义表头、核心内容、表尾
