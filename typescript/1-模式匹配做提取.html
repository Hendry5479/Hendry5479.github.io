# 模式匹配

通过extends做类型参数匹配

结果保存到infer声明的局部类型变量中

如果匹配成功就从变量中提取出类型

# 数组类型

# First、Last等等

提取数组的第一个元素的类型

type getFirstFromArr<A extends unknown[]> = A extends [infer fir, ...unknown[]] ? fir : never;

A通过extends约束为只能是数组类型

定义unknown，数组元素类型可以是任何值

unknown接收，any除了接收还能赋值

# 字符串类型

把需要提取的部分放到 infer 声明的局部变量里

# StartsWith

判断字符串是否以某个前缀开头

type StartsWith<Str extends string, Prefix extends string> = Str extends `${Prefix}${string}` ? true : false;

# Replace

替换 replace from to

type ReplaceStr<Str extends string, From extends string, To extends string> =  Str extends `${infer Prefix}${From}${infer Suffix}` ? `${Prefix}${To}${Suffix}` : Str;

# Trim

替换空格，需要逐个匹配，递归

type TrimRight<Str extends string> = Str extends `${infer Rest}${' ' | '\n' | '\t'}` ? TrimRight<Rest> : Str;
type TrimLeft<Str extends string> = Str extends `${' ' | '\n' | '\t'}${infer Rest}` ? TrimLeft<Rest> : Str;

type Trim<Str extends string> = TrimRight<TrimLeft<Str>>

type TrimResult = Trim<' ab '>

# 函数

GetParameters

type GetParameters<Func extends Function> = Func extends (...args: infer Args) => unknown 
  ? Args : never;

参数类型放到用 infer 声明的局部变量 Args 里，返回值可以是任何类型，用 unknown

# GetReturnType

type GetParameters<Func extends Function> = Func extends (...args: infer Args) => unknown ? Args : never;

type param = GetParameters<(name: string, age: number) => string>

type GetReturnType<Func extends Function> = Func extends (...args: any[]) => infer ReturnType ? ReturnType : never;

type returnType = GetReturnType<(name: string, age: number) => string>

# GetThisParameterType

type GetThisType<T> = T extends (this: infer ThisType, ...args: any[]) => any 
  ? ThisType : unknown;
